## 一、项目定位

AquaVir-KB 是一个面向**水生无脊椎动物病毒**的综合性数据库，覆盖甲壳类、软体动物、刺胞动物等 8 个宿主门类，整合病毒分类、宿主关联、文献证据、蛋白注释、地理分布和系统发育数据，目标投稿 **NAR Database Issue**。

## 四、近期已完成工作（5月-6月初）

1. **数据清洗与去重**
   - 清理 103 条非水生错配病毒，标记为 `non_target`
   - 处理 14 条 ICTV VMR 数字占位条目，合并到正式物种
   - 修复 69 条 minor phyla 错配

2. **文献与证据回填**
   - 零文献病毒从 574 → **373**（主要是宏基因组来源，无法完全清零）
   - 零证据病毒从 61 → **2**
   - ICTV 映射从 36 → 445，VMR 映射从 95 → 524

3. **地理信息补全**
   - 从 1,981 → 4,326 条分离株有 country（+2,345）
   - 通过 NCBI GenBank XML 解析补充

4. **技术基础设施**
   - FastAPI 后端修复 3 个崩溃端点
   - 移除 Sci-Hub 相关代码与文件（合规清理）
   - 修复 SQL 注入风险、更新 LICENSE 为 CC BY 4.0
   - 生成 6 张论文图表（RdRp 树、地理分布、桑基图等）

5. **NAR 投稿准备**
   - 完成全面审计（43 项问题，8/9 Critical 已修复）
   - 撰写 DATA_AVAILABILITY、NOVELTY_COMPARISON、SUSTAINABILITY 等文档
