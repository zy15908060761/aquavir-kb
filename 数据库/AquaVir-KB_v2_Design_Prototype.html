Design Comparison

What changes and why

CURRENT
✗
Information Overload
Detail page shows ALL content at once (~3,000 words/screen). Users cannot find key info.
✗
No Dark Mode
NAR reviewers often work at night. Bright white screen causes eye strain.
✗
Single Color Theme
Everything is Teal. Genome, host, protein, geography all look the same. No visual hierarchy.
✗
No Bulk Actions
Search results cannot be downloaded in batch. Each record must be accessed individually.
✗
Vertical Dashboard Stack
Home page is a long scroll of cards. No visual rhythm, no data storytelling.
NEW v2.0
✓
Progressive Disclosure
Layer 1: Hero summary (key data only). Layer 2: Topic tabs (lazy load). Layer 3: Inline expand.
✓
Dark Mode
System preference detection + manual toggle. Full dark palette (Slate-900 base, not inverted).
✓
Semantic Color System
Genome = Blue, Host = Green, Protein = Purple, Virulence = Orange/Red, Geography = Cyan.
✓
Bulk Download Bar
CSV / FASTA / JSON download buttons on every search result page. Standard for NAR databases.
✓
Bento Grid Homepage
Modular card grid. Each data type has its own visual block. Map thumbnail + chart preview.
