# Data Use Agreement

CrustaVirus DB is intended for research, education, and reproducible database publication review.

## Permitted Use

Users may browse, download, analyze, and redistribute the release-filtered exports under CC BY 4.0, provided that CrustaVirus DB and original source accessions/literature are cited.

## Scientific Scope

Database records may include release-filtered target records, extended ecological context, non-target host/sample records, host-genome artifacts, and unreviewed candidate evidence. Publication-quality analyses should use the release analysis views and public exports, not raw tables, unless the raw scope is explicitly justified.

## Biosafety

The database contains sequence and literature metadata. It is not a protocol repository and does not provide instructions for culturing, propagating, engineering, or experimentally modifying pathogens. Users must comply with applicable biosafety, biosecurity, institutional, and legal requirements.

## No Warranty

The database is provided as-is. Release-filtered fields are maintained with provenance where available, but users should verify records against original accessions and publications before regulatory, clinical, aquaculture-management, or high-impact scientific use.
