DATABASE URL: https://aquavirdb.com

DESCRIPTION:
AquaVir-KB is a comprehensive, evidence-driven knowledge base for viruses infecting aquatic invertebrates. It currently integrates:

- 1,601 virus species spanning 9 aquatic invertebrate phyla (Arthropoda, Mollusca, Cnidaria, Echinodermata, Porifera, Annelida, Platyhelminthes, Rotifera, Nematoda)
- 17,867 virus isolates with standardized metadata
- 353,160 literature-traceable evidence records with 92% PMID / 90% DOI coverage
- 29,004 annotated protein sequences with InterPro (88.5%), GO (78%), and KEGG (37%) functional annotation
- 74.3% ICTV classification coverage (1,189 of 1,600 active viruses mapped to official taxonomy)
- 9,065 curated references from 1,016 journals

The database provides an interactive web interface with faceted search, BLAST against RdRp proteins, phylogenetic trees, geographic distribution maps, a host-virus interaction network, and a REST API (67 endpoints with Swagger documentation). All data is freely downloadable in FASTA, XLSX, CSV, and TSV formats under CC BY 4.0.

DIFFERENTIATION FROM EXISTING RESOURCES:
Unlike NCBI Virus (which provides raw sequence records), ICTV VMR (exemplar-based taxonomy only), or IMG/VR (metagenome-focused), AquaVir-KB uniquely:
1) Integrates virus genomics with host ecology, geographic distribution, and graded literature evidence in a single queryable framework
2) Covers ALL aquatic invertebrate phyla where existing invertebrate virus databases are phylum-restricted or nonexistent
3) Provides evidence-level grading (high/medium/low) for each virus-host association, making it possible to distinguish confirmed infections from metagenomic co-occurrence

The database fills a recognized gap: while terrestrial livestock and human viral diseases are well-served by existing databases, aquatic invertebrates — critical to global aquaculture and marine ecosystem health — lack a dedicated knowledge base. With aquaculture providing >50% of global seafood and viral diseases causing billions in annual losses, AquaVir-KB addresses a pressing need.

The website is free and open to all users with no login requirement, served over HTTPS, and committed to maintenance for a minimum of 5 years.
